# API Project

This directory contains Python scripts that interact with the JSONPlaceholder REST API.

## Tasks

### 0. Gather data from an API
A Python script that returns information about an employee's TODO list progress.

### 1. Export to CSV
Exports an employee's TODO list data to a CSV file.

### 2. Export to JSON
Exports an employee's TODO list data to a JSON file.

### 3. Dictionary of list of dictionaries
Exports TODO data for all employees in a single JSON file.

## Requirements
- Python 3
- `requests` module
